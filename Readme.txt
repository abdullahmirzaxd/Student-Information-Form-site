# Student Info Form using Streamlit

This is a simple web app built using **Streamlit** that allows students to enter their information (Name, Father's Name, Address, and Semester) through a form.

## Features:
- Collects user information through input fields
- Displays entered data upon form submission

## How to Run
1. Install the necessary dependencies using:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app using:
   ```bash
   streamlit run your_app.py
   ```

## Requirements:
- Python 3.x
- Streamlit
